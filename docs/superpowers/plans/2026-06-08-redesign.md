# IDEAL Hotel — Frontend Redesign Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Redesign all 7 pages of the IDEAL Hotel static site to a boutique luxury aesthetic: dark walnut hero sections, centered blur-glass nav, split homepage hero, amber accents, warm cream content sections, and calm hotel UX throughout.

**Architecture:** CSS-first — add dark palette tokens and new component classes to `styles.css`, then update HTML on all pages to use the new structures. `site.js` gets one small change to the nav scroll logic. No build tools; all changes are to static files.

**Tech Stack:** Vanilla HTML/CSS/JS. Fonts: Cormorant Garamond + Mulish (no change).

- Modify: `Design/assets/site.js` (lines 76–85, header scroll section)

```javascript
  /* ---- header scroll state ---- */
  var header = doc.querySelector('.site-header');
  if (header) {
    var onScroll = function () {
      header.classList.toggle('is-solid', window.scrollY > 40);
    };
    onScroll();
    window.addEventListener('scroll', onScroll, { passive: true });
  }
```
